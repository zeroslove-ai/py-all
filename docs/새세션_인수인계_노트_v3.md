### 2-3. ⚠️ 중요 제약
- **이 키로는 REST API만 가능** (`table()`, `rpc()`)
- **Raw SQL (`SELECT * FROM ...`) 직접 실행 불가**
- PostgreSQL 직접 연결(`psycopg2`)에는 **별도 비밀번호** 필요 (Supabase Dashboard → Settings → Database)
- Supabase REST는 보안상 raw SQL 차단 (PostgREST 레이어)
