## 1. 아키텍처 전체

```
[사용자] → Dify 앱(udify.app) → advanced-chat 워크플로우
   ├─ Supabase RPC (get_context/save_turn/set_master/patch_save/debug_log 등)
   ├─ Supabase Storage (Image 버킷 - 캐릭터 이미지)

        → presigned URL(r2.cloudflarestorage.com) → Dify <audio> 인라인 재생
```

**핵심 제약(절대 규칙):**
1. deliverable은 TXT 말고 **YML**
2. 보안/API키 관련 이슈 제기 금지 (사용자 지시)
3. 서사 LLM(`c_llm`)과 답변(`c_answer_stream`) **사이에 코드 노드 금지** — 스트리밍 최우선
4. 이미지는 **최종 렌더러가 항상 한 장 붙이는 B안** (억제 가드 넣으면 heroine2/3 엇갈린 한 장 사고 재발)

### 현재 배포된 Worker 소스 (번들 전 원본)
```js
import { AwsClient } from 'aws4fetch';

export default {
  async fetch(request, env) {
    if (request.method !== 'POST') {
      return Response.json({ ok: false, stage: 'method', status: 405 }, { status: 405 });
    }
    try {
      const { text, voice_id, key } = await request.json();
      const fish = await fetch('https://api.fish.audio/v1/tts', {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${env.FISH_API_KEY}`,
          'Content-Type': 'application/json',
          'model': 's2.1-pro-free'   // S2계열 필수 — [대괄호] 연기태그는 S2에서만 작동
        },
        body: JSON.stringify({ text, reference_id: voice_id, format: 'mp3' })
      });
      if (!fish.ok) {
        const body = await fish.text();
        return Response.json({ ok: false, stage: 'fish', status: fish.status, body: body.slice(0, 500) });
      }
      const mp3 = await fish.arrayBuffer();
      const objectKey = key || `${crypto.randomUUID()}.mp3`;
      await env.tts.put(objectKey, mp3, { httpMetadata: { contentType: 'audio/mpeg' } });

      const r2 = new AwsClient({
        accessKeyId: env.R2_ACCESS_KEY_ID,
        secretAccessKey: env.R2_SECRET_ACCESS_KEY,
        service: 's3', region: 'auto'
      });
      const signUrl = new URL(`https://98efff3e9faacb9e57a14177682143a8.r2.cloudflarestorage.com/tts/${objectKey}`);
      signUrl.searchParams.set('X-Amz-Expires', '3600'); // 최대 604800(7일)까지 가능
      const signed = await r2.sign(signUrl, { method: 'GET', aws: { signQuery: true } });

      return Response.json({ ok: true, url: signed.url, key: objectKey });
    } catch (e) {
      return Response.json({ ok: false, stage: 'exception', message: String(e && e.message || e).slice(0, 300) }, { status: 500 });
    }
  }
};
```

### c_code_voice_lookup (v38/v39 현행 코드 — 실데이터 검증 완료)
```js
function main({ctx}) {
  let obj = {};
  try { obj = JSON.parse(ctx || '{}'); } catch (e) {}
  const save = obj.save || {};
  const mems = Array.isArray(obj.recent_memories) ? obj.recent_memories : [];
  const last = mems.length ? mems[mems.length - 1] : null;
  const raw = (last && last.content) ? String(last.content) : '';
  const cid = save.last_character_id || '';
  const master = obj.master || {};
  const chars = master.characters || {};
  const name = (chars[cid] && chars[cid].name) ? chars[cid].name : '';

  // 서사 본문만 사용 (--- 구분선 이후 마인드모니터/상황판은 TTS 대상 아님)
  const scene = raw.split('\n---')[0];

  // 화자 태그: (**)이름(**) (지문)?: "대사" → 지문은 [대괄호] 연기 태그로 변환
  const speakerRe = /\*{0,2}([가-힣A-Za-z0-9]{2,12})\*{0,2}\s*(?:\(([^)\n]{0,60})\))?\s*:\s*"([^"]+)"/g;
  let lines = [];
  let m;
  while ((m = speakerRe.exec(scene)) !== null) {
    const speaker = m[1].trim();
    if (name && speaker !== name) continue;
    const dir = (m[2] || '').trim();
    lines.push((dir ? '[' + dir + '] ' : '') + m[3]);
  }

  let text = lines.length
    ? lines.join(' ... ')
    : scene.replace(/\*{0,2}[가-힣A-Za-z0-9]{2,12}\*{0,2}\s*(?:\([^)\n]{0,60}\))?\s*:\s*"[^"]+"/g, '').slice(0, 200).trim();

  // JSON body 깨짐 방지
  text = text.replace(/["\\]+/g, ' ').replace(/[\r\n]+/g, ' ').replace(/\s{2,}/g, ' ').trim();

  let voice = (chars[cid] && chars[cid].voice_id) ? chars[cid].voice_id : '';
  if (!voice) {
    const first = Object.keys(chars)[0];
    if (first && chars[first].voice_id) voice = chars[first].voice_id;
  }
  return { tts_text: text, voice_id: voice };
}
```

### 신음 규칙 (moan_rules — 새게임 프로젝트 원본, 현행 게임 characters.신음타입에도 있음)
- 단계: 참음(순응0~30) 윽.../큭 → 누출(30~50) 하앗/읏 → 연속호흡(50~70) 아/앙! → 완전굴복(70~100) 흐아아…/아아앙!
- 부호 규칙: `...`=참음/저항, `?!`=당황/누출, `!~`=호흡, `! ❤️`=굴복
- **미적용 아이디어(보류)**: 장면 수위 감지 + 신음 인터리브로 H씬 낭독 강화 — 코드 노드 낸 문자열 매칭이라 지연 1ms 미만이지만, 사용자가 "설계에 없는 내용"으로 보류 처리

### 3-1. YML 버전 저장소 (yml_versions) — 새 세션 접근 방법

- 프로젝트: `ckzwlmoojtmcpwlqsqzv` (채팅플로우)
- **content = YML 원문 텍스트 그대로** (압축 없음). `extensions.http` 확장 활성화됨(2026-07-19)
- 적재된 버전: **v35, v36, v37_최종, v38, v39** — 전부 로컬 파일과 md5 일치 검증 완료

```sql
SELECT version, filename, note, created_at, length(content), md5(content)
FROM yml_versions WHERE app='게임빌더_v1' ORDER BY created_at DESC;
```

```sql
-- 2) DB가 직접 페치해서 저장
SET http.timeout_msec = 90000;
INSERT INTO yml_versions (app, version, filename, content, note)
SELECT '게임빌더_v1', 'v40', '게임빌더_v40.yml', h.content, '메모'
FROM extensions.http_get('<붙여넣기 URL>') AS h
ON CONFLICT (app, version) DO UPDATE SET filename=EXCLUDED.filename, content=EXCLUDED.content, note=EXCLUDED.note;
-- 3) md5(content)가 로컬 파일 md5와 일치하는지 반드시 검증
```

## 8. 검증된 사실 (실측 근거 있음)

1. **Dify CSP media-src**: `self, data:, blob:, *.dify.ai, *.udify.app, *.r2.cloudflarestorage.com` 허용 / SoundHelix, Supabase, `*.r2.dev`, `*.workers.dev` **차단** → 차단 도메인은 플레이어만 뜨고 0:00
2. `<audio controls>` 태그는 Dify 마크다운에서 렌더됨 (upload.dify.ai 비프음 재생으로 실증)
3. base64 data URL은 재생바 자체가 안 뜸 → 비현실적
4. Dify http-request 노드의 `body` 출력은 문자열 → `JSON.parse(body)` 필요
5. presigned URL `X-Amz-Expires` 최대 604800초(7일)
6. Cloudflare **API 배포 시 metadata.bindings 누락하면 기존 R2 바인딩 삭제됨** (실제 사고→복구). secret_text는 값 없이 이름만 넣으면 기존 값 유지(PATCH /settings도 가능)
7. 대시보드 에디터는 npm import를 자동 번들링하지만, **API 배포는 안 해줌** → esbuild 필수
8. recent_memories는 오름차순, `rn` 필드 없음 → idx 기반 압축 금지
9. 이미지 선택: image_id 직접 → character_id+emotion_id fallback → default
10. get_context로 난 저장 오염은 없었음. 이중이미지는 LLM 직접 출력 + 최종 렌더러 중복이 원인
