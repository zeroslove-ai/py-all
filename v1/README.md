# 게임빌더 v1 보관함

운영 중인 v2와 분리해 둔 이전 프로젝트 자료입니다. 이 폴더의 파일은 기록·참고용이며 현재 배포 대상으로 사용하지 않습니다.

| 경로 | 내용 |
| --- | --- |
| `dify/yml/` | Dify 워크플로우 버전 및 변경 이력 |
| `tts_worker/` | 기존 TTS Cloudflare Worker와 배포 스크립트 |
| `docs/` | v1 인수인계·이미지 카탈로그 |
| `external/` | Grok·Kimi 작업 사본과 로그 |
| `tools/` | 이전 프록시 배포 도구 |
| `legacy_notes/` | 이전 Supabase 연결 예시 |

운영 코드는 저장소 루트의 `pages/`, `worker/game-proxy-v2.js`, `supabase/`, `scripts/deploy-worker.mjs` 및 `docs/project_v2/`를 기준으로 합니다.
